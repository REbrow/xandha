# Project Name
Xandha Tourism Web Application
## Installation
1. Simply Fork or Clone
2. Run "npm Install"
3. Run "bower Install"

## Usage
1. Run "gulp serve"
2. For production build, simply run "gulp"
3. To test production build, run "gulp:dist"
